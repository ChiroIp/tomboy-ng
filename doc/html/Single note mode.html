Single note mode

Opens, in standalone mode, a note. Does not check to see if another copy of tomboy-ng is open but will prevent a normal startup of tomboy-ng. Won't interfere with an existing copy, however. If the note you specify doesn't exist, it will be created. As well as .note files, single note mode also opens .txt and .rtf files. However, if edited they will be saved as duplicates with a .note extension, as to preserve XML tags when next loaded. Linux variant tomboy-ng -o path/to/file.note Mac variant open tomboy-ng.app --args -o path/to/file.note
